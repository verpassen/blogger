為什麼要加 "if __name__ == '__main__'"

Sep 7 2021 Python 2 minutes read (About 336 words)

為什麼要加這一段?

Python中每個script可以單獨被執行，也可以被當成模組(module)引用。
而當被直接執行時， __name__ 這個特徵值 被改成 “__main__“。
不然，就會被當成某個被引用的模組名稱

為什麼這樣設計?

在 if name == “__main__“:
這一段後的程式如果會被執行
表示目前的這個檔案是被當作主程式在執行
因此，後面放的部分，是當目前這個檔案在被當成模組匯入時，不想被執行的某段程式或功能

目前常看到的架構

所以當這個檔案直接被執行，就會執行main這個函數。
如果被引用，則什麼都沒做，不過其他script還是可以使用這個檔案中所定義的類別(class)和函數(function)

BT vs. CAT(doing)

Jun 26 2021 Standard 2 minutes read (About 333 words)

定義

標題所提到的BT 與 CAT 都是CNC 銑床中會使用到的刀柄系統。
其中 BT 為日本所訂的標準，CAT 則為美國，當然還有其他的系統規格，
例如: HSK、PSC(Capto)、BBT 等

比較這些不同的系統，可以寫一大篇了。
這篇就先比較BT和CAT，其他的就後續再補上

刀柄相關規範

  • CAT - ASME B5.50-2009
  • BT - JIS B6339-1986

其他

  • DIN69871
  • DIN6391 關於用於刀柄上筒夾的相關規定

差異點

  1. BT的斜錐面長度較CAT來的短，差約4mm ，因此使用的拉釘規格也有所不同

  2. BT的法蘭厚度較厚，換刀機構的夾持fork也會需要變更

  3. BT的拉釘鎖固的螺牙是公制的，CAT則是英制

Face_Grooving

Jun 1 2021 Cutting tools 3 minutes read (About 518 words)

Face grooving 的 DAXN和 DAXX 的分法

軸向的插槽，會有兩種干涉的情況
一是刀柄曲率半徑過小，與內圓干涉
一是刀柄曲率半徑過大，與外圓干涉


在ISO規範種，對於槽刀可以應用的加工範圍有所規定
DAXX 和 DAXN


右上的圖，放大來看，就是標示這樣


不過實際上使用，還是曾經誤解過 DAXN 和DAXX 的定義方式
端面切槽刀實際使用範圍(Application range)

如果 [AR > DAXX] :
槽加工 只能往遠離圓心方向移動 (往半徑大的方向)
或是 [ AR < DAXN]
槽加工只能往圓心移動(往半徑小的方向)
另外可能 (也就是 DAXX > AR > DAXN)
槽加工左右都可以


觀察槽刀的形狀，外圈曲率較小，內圈曲率較大。
因此使用上，在大於DAXX 的部分，可以向外移動
在小於DAXN 的部分，可以向內移動不干涉

舉個例子
下圖的槽刀 ，型錄上所標示的最小容許加工直徑DAXN 為55，最大容許加工直徑DAXX 為75
以2D繪製干涉區域，分別為54.28的圓和 70
先看到往中心方向橫向加工，如果加工直徑小於70 mm，大於55 mm
這個範圍，刀具若是再往中心移動會先碰觸到刀桿的內圓(干涉)
刀具往外移動，則會先碰觸刀桿的外緣，一樣是干涉
因此若要橫移加工，在[55=<D=<70]範圍內都是OK的


結論

- DAXX 和 DAXN 的區分方式
如果 [加工直徑 > DAXX] :
    槽加工 只能往外移動 (往半徑大的方向)
或是 [ 加工直徑 < DAXN] 
    槽加工只能往內移動(半徑小的)
另外可能 (也就是 DAXX > 加工直徑 > DAXN)
    槽加工左右都可以
- 用3D 確認一下干涉，當然還是以實物為準。
    曾經用3D抓出來的數據跟型錄建議值有點差異。

Thread Pitch Vs. Threads per inch

TPI (thread per pitch) - 每英寸有幾牙

thread per pitch 簡稱 TPI ，常見於鎖固件的標稱(註.1)

  1. 單位: 英制(US)
  2. 格式: 1/4”-20X2”

1/4” 代表螺紋直徑為1/4英吋
每英吋20牙
螺紋長度總長2英吋

thread pitch 牙距

  1. 單位: 公制(Metric)
  2. 格式: M2X0.4X5M

M2代表螺紋直徑2 公釐(mm)
0.4 螺距
螺紋長度總長5 公釐 (mm)

TPI 與 Thread Pitch 的換算

舉例 TPI = 24 的 螺紋 相當於 thread pitch 多少?
每英寸有24 牙，
一英吋 = 25.4 mm
換句話說，每25.4 mm 有24 牙
也就是牙距為 25.4/24 ~ 1.058 mm
